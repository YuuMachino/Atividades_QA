# ATIVIDADES QA

Repositório destinado ao armazenamento das atividades desenvolvidas na disciplina de Qualidade de Software (QA).

## Organização

As atividades estão organizadas em pastas separadas de acordo com o que foi solicitado ou a data de realização do exercício.
Cada pasta contém os arquivos correspondentes à atividade realizada no dia.

## Objetivo

Registrar e organizar as atividades práticas da disciplina, incluindo análises, requisitos e casos de teste.
